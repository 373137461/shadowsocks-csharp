Shadowsocks for Windows
=======================

从原Shadowsocks项目（https://github.com/shadowsocks/shadowsocks-csharp）分支，按照个人需要改变。

#### 改变的内容

1. 按照原作者的意思，启用系统代理的同时会打开SOCK5代理服务，改变后的策略是：启用SOCK5代理服务，但是不会修改系统代理配置。系统代理配置即IE的代理配置。
